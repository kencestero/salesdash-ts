MCP SETUP - 5 STEPS
====================

STEP 1: Get GitHub Token
-------------------------
1. Go to: https://github.com/settings/tokens/new
2. Name: "Claude MCP"
3. Check boxes: repo, read:org, read:user
4. Click "Generate token"
5. COPY THE TOKEN (you won't see it again!)


STEP 2: Copy Config File
-------------------------
Open PowerShell, run this:

mkdir "C:\Users\kence\AppData\Roaming\Claude" -Force
copy "C:\Users\kence\salesdash-ts\claude_desktop_config.json" "C:\Users\kence\AppData\Roaming\Claude\claude_desktop_config.json"


STEP 3: Edit Config File
-------------------------
Open the file:

notepad "C:\Users\kence\AppData\Roaming\Claude\claude_desktop_config.json"

Replace:
- YOUR_GITHUB_TOKEN_HERE → paste your GitHub token
- YOUR_NEON_DATABASE_URL_HERE → paste from your .env file
- YOUR_BRAVE_API_KEY_HERE → skip this (remove the whole brave-search section if you want)


STEP 4: Restart Claude
-----------------------
Close Claude Desktop completely
Open it again


STEP 5: Test It
---------------
Ask Claude: "What MCP tools do you have?"
